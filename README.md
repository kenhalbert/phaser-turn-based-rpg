# phaser-turn-based-rpg
A very incomplete turn-based RPG built with the Phaser game engine.

[Play it in your browser here.](https://kenhalbert.github.io/phaser-turn-based-rpg/)

Based on [this tutorial](https://gamedevacademy.org/how-to-create-a-turn-based-rpg-game-in-phaser-3-part-1/).

# Controls
## World mode
Move: WASD

## Battle mode
Change menu selection: arrow keys

Confirm selection: space
